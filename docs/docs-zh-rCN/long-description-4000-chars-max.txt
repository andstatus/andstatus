AndStatus 是一个开源的低流量社交网络客户端
具有树状线程对话。
它支持不同的社交网络,
包括
GNU社交 (例如Quitter.se, LoadAverage, Vinilox 等), Twitter 和 Pump.io.
AndStatus 可以将来自所有网络的提要合并到一个时间轴中
即使在离线状态下也可以阅读和发布。

适用于 Android v.7.0+ 设备。

AndStatus 的区别特征:

1.
 许多帐户在不同的社交网络。 您可以在每个帐户中有多个帐户,
以任何“您”的身份进行书写/回复,
并在帐户和系统之间进行分享。
2. 您无需在线阅读时间表并发布您的更新:
即使在重新启动后,
草稿和未发送的帖子也会保留。 他们将在您的设备在线时发送。
3. 便利的树状“对话视图”。
4. “全球搜索”允许您在一个查询中搜索所有网络中的公共消息。
5. 作为用户列表和时间线 (以每个用户的最新消息) 呈现的朋友和追随者列表
6. 推文/消息/头像/附加图片在后台同步,
当你的设备有一个很好的连接。 默认情况下，附件只能通过WiFi下载。
7. 缓存的数据可能会被保存多年或几天 - 这是您的选择。
8. 备份和恢复您的设备之间的帐户和消息。 拥有你的数据！
9. 免费, 没有广告, 社区支持。
