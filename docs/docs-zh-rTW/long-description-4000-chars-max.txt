AndStatus 是一個開源的低流量社交網絡客戶端,
具有樹狀線程對話。
它支持不同的社交網絡,
包括
GNU社交 (例如Quitter.se, LoadAverage, Vinilox等), Twitter 和 Pump.io.
AndStatus 可以將您的提要從所有網路合併到一個時間表中,
即使在離線狀態下也可以閱讀和發布。

適用於Android v.4.1+設備。

AndStatus 的區別特徵:

1. 許多帳戶在不同的社交網絡。 您可以在每個帳戶中有多個帳戶,
以任何“您”的身份進行書寫/回复,
並在帳戶和系統之間進行分享。
2. 您無需在線閱讀時間表並發布您的更新:
即使在重新啟動後,
草稿和未發送的帖子也會保留。 他們將在您的設備在線時發送。
3. 便利的樹狀“對話視圖”。
4. “全球搜索”允許您在一個查詢中搜索所有網絡中的公共消息。
5. 作為用戶列表和時間線 (以每個用戶的最新消息)
呈現的朋友和追隨者列表
6. Tweets/消息/頭像/附加圖片在後台同步, 當你的設備有一個很好的連接。 默認情況下, 附件只能通過WiFi下載。
7. 緩存的數據可能會被保存多年或幾天 - 這是您的選擇。
8. 備份和恢復您的設備之間的帳戶和消息。 擁有你的數據！
9.
 免費, 沒有廣告, 社區支持。
