AndStatus Project Home Page

Welcome to the AndStatus project - creator of the AndStatus application - low traffic social networking client with tree-like threaded conversations for Android.

The project was created in December 2011 as a community initiative, not associated with any person or commercial entity (so andstatus.org domain was registered) and not tied to Twitter only (the application name changed from AndTweet to AndStatus to avoid possible Twitter ban). Source code hosting moved to the GitHub, new AndStatus "GitHub organization" created to facilitate the team growth. The organization is the owner of the AndStatus source code repository.

Permanent AndStatus project address (this page): http://andstatus.org
